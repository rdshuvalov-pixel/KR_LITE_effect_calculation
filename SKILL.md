# Merge Entersite Reports (склейка периодов)

Объединяет набор товарных Excel-отчётов Энтерсайт в один файл с подённой разбивкой себестоимости и продаж. **Склеивает периоды** с приоритетом более позднего файла при пересечениях.

## Source Files

Входные файлы — xlsx из папки `данные Энтерсайт/`. Формат имени всегда содержит период:

```
*_с_ДД_ММ_ГГГГ_по_ДД_ММ_ГГГГ_*.xlsx
*_с_ДД.ММ.ГГГГ_по_ДД.ММ.ГГГГ_*.xlsx
```

Поддерживаются оба формата (подчёркивания или точки между день.месяц.год).

### Структура листа (лист "My Sheet")

| Колонка | Содержание |
|---------|-----------|
| B | Артикул (product_id) |
| C | Товар (name) |
| L | Реализация: Количество |
| M | Реализация: Закуп (сумма себестоимости) |
| N | Реализация: Розница (выручка) |

Строка 2 — итоговая (без артикула), пропускать. Данные товаров — со строки 3.

## Output File

Итоговый файл: `данные Энтерсайт/Итоговые_данные_Энтерсайт.xlsx`

### Лист "Себестоимость"

| Колонка | Формула |
|---------|---------|
| product_id | Артикул |
| stock | Всегда `1` |
| cost | `Реализация: Закуп / Реализация: Количество` (за единицу) |
| date | Каждый день периода (YYYY-MM-DD) |

### Лист "Продажи"

| Колонка | Формула |
|---------|---------|
| product_id | Артикул |
| name_full | Товар |
| quantity | `Реализация: Количество / кол-во дней в периоде` |
| recorded_on | Каждый день периода (YYYY-MM-DD) |
| price | `Реализация: Розница / Реализация: Количество` |

## Overlap Resolution

Файлы обрабатываются **по возрастанию даты начала** периода.  
При пересечении периодов **более поздний файл перезаписывает** данные раннего на пересекающихся датах.

Пример:
- Файл A: 02.12–08.12 → записывает дни 02–08.12
- Файл B: 07.12–15.12 → перезаписывает 07–08.12 и добавляет 09–15.12

Результат: 02–06.12 из файла A, 07–15.12 из файла B.

## Skip Rules

Пропускать строки, где:
- Артикул пустой или `None`
- Реализация: Количество = 0 или `None`
- Имя файла начинается с `~$` или `Итог`

## Running

```bash
python3 scripts/generate.py [--data-dir PATH] [--output PATH]
```

По умолчанию: `--data-dir` = `данные Энтерсайт/`, `--output` = `данные Энтерсайт/Итоговые_данные_Энтерсайт.xlsx`

**UI:** Страница «Объединение Энтерсайт» в Streamlit (sidebar).

## Testing

```bash
python3 scripts/test.py
```

Тесты проверяют:
1. Расчёт себестоимости (cost = закуп/количество)
2. Расчёт продаж (quantity = количество/дней, price = розница/количество)
3. Приоритет при пересечении периодов
4. Пропуски дат между файлами
5. Одинаковое число строк на обоих листах
6. Конкретные значения по нескольким артикулам
